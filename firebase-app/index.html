CamiPi

Plataforma de demonstração com autenticação, modelagem em árvore JSON e permissões separadas entre user e admin com Firebase Realtime Database.

Autenticação

Cadastro e login com Firebase Auth usando e-mail e senha.

Segurança

Acesso controlado com Firebase Security Rules por tipo de cargo.

Modelagem

Estrutura JSON com nós principais /users e /admin-data.

Acesso

Entrar na aplicação

Inicialize o Firebase e faça login para ser redirecionada ao painel correspondente ao cargo.

Firebase ainda não inicializado.
Nenhum usuário autenticado.
